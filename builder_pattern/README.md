# 建造者模式



## 1. 概念

建造者模式是一种对象构建模式，是将一个复杂对象的构建与它的表示分离，使得同样的构建过程可以创建不同的表示。



## 2. 理解

当要构建的对象很大并且需要多个步骤时，使用构建器模式，有助于减小构造函数的大小。
